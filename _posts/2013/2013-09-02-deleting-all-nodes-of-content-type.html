--- layout: post title: "Deleting all nodes of a content type using drush and devel" date: 2013-09-02 08:50:00 source: pblog tags: ["drupal 6", "drupal 7", "drush", "drupal"] redirect_from: - "/2013/09/deleting-all-nodes-of-content-type.html" --- While developing the open source data logger (started here but expanded to post from Arduino to Drupal) the need to regularly delete all content of a given type arises fairly regularly. This can be done through the user interface with the Devel module but it is much quicker to use Drush:

 drush genc 0 --kill --types=article
